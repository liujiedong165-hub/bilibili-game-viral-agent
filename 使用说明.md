# 使用说明

## 第一步：启动虚拟环境

```powershell
.\.venv\Scripts\activate
```

## 第二步：配置 API Key

复制 `.env.example` 为 `.env`，填写：

```env
OPENAI_API_KEY=你的APIKey
OPENAI_MODEL=gpt-5.4-mini
```

## 第三步：采集数据

```powershell
python -m app.collect
```

## 第四步：分析数据

```powershell
python -m app.analyze
```

## 第五步：运行 Agent

```powershell
python -m app.agent
```

报告会生成在：

```text
data/reports/
```

Dashboard 会生成在：

```text
data/dashboard.html
```
